Endpoints Sabrina API TP Final

    /api

  • POST: /api/register (receives, body: { email, password, username })
  • POST: /api/login (receives, body: { email, password })
  • POST: /api/products/new (receives, body: { userId, status, availableQty, price, name })
  • POST: /api/cart/add (receives, body: { cartId, productId, quantity, price, status })
  • GET: /api/usuarios/:id/ordenes (receives, params: id)
  • GET: /api/usuarios/:id/cart (receives, params: id)
  • GET: /api/products
  • PUT: /api/cart/:id/purchase (receives, params: id)
